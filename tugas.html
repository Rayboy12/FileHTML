Perkenalkan nama saya Arnoldew Ray Ruby,saya bisa di panggil Ray juga bisa di panggil Arnold Saya sekarang tinggal di Kabupaten Malang dan alamat kost saya yaitu di Jl. Kapi Sraba X No.8. Keduyo, Mangliawan, kec, Pakis saya cuma 2 bersaudara saya dan kakak saya, yang sekarang berkuliah di surabaya, lebih tepatnya di kampus UPN Veteran Jawa Timur saya berasal dari keluarga blasteran disebabkan ayah saya seorang imigram Philipina dan ibu saya berasal dari Indonesia, tetapi sekarang bekerja di pabrik minyak sawit, yang berada di Sabah, Malaysia.

Saya mendapatkan kesempatan untuk SMA di jawa dengan beasiswa full dari pemerintah, sudah sekitar 3 tahun lebih saya di jawa dan sekarang saya berkempatan lagi untuk kuliah dengan beasiswa lagi di STIMIK PPKIA Pradnya Paramita, dan saya mengambil jurusan S1 Sistem Informasi.

Itu saja dari saya Arnoldew Ray Ruby, semoga tugas saya bisa dapat nilai tinggi amin
